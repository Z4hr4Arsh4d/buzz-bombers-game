# Buzz Bombers Game (C++ / SFML)

A arcade-style recreation of the classic **Buzz Bombers** game built using C++ and SFML.
This project demonstrates core game development concepts including movement, collision handling, enemy behavior, and level progression.

---

## Features

* Player movement and shooting system
* Worker bees and killer bees with different behaviors
* Honeycombs generated after defeating bees
* Hummingbird interaction system
* Flower obstacles affecting gameplay
* Multiple levels with increasing difficulty
* Score tracking system

---

## Technologies Used

* C++
* SFML (Simple and Fast Multimedia Library)

---

## How to Run

1. Install SFML on your system
2. Clone the repository:

   ```
   git clone https://github.com/Z4hr4Arsh4d/buzz-bombers-game.git
   ```
3. Open the project in your C++ environment
4. Compile and run the program

---

## Gameplay Overview

* Control the spray can to shoot bees
* Worker bees turn into **yellow honeycombs**
* Killer bees turn into **red honeycombs**
* Avoid over-spraying (it can affect gameplay)
* Clear levels by defeating bees and managing obstacles

---

## Controls

* Move left/right
* Spacebar → Shoot

---

## Project Structure

* `buzz.cpp` → Main game logic
* `score.txt` → Score tracking file
* `resources/` → Images, sounds etc
